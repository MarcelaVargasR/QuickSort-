# Overview

For this project, I built a TypeScript application that demonstrates how to organize logic using classes, process lists of data, apply recursion, work with asynchronous functions, and safely handle exceptions.

This software is a QuickSort analyzer written in TypeScript. It loads lists of numbers asynchronously, validates the input, sorts the numbers using a recursive QuickSort algorithm, and displays useful statistics in the terminal such as the original list, sorted list, total count, smallest number, largest number, and average.
